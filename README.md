## 📖 项目简介

ConnectionClient 是一个基于 Qt 框架和 MySQL 数据库开发的现代化即时通讯客户端。它提供了完整的社交通讯功能，包括用户管理、好友系统、实时聊天、文件传输等核心特性。项目采用客户端-服务器架构，支持多用户并发通信。

## ✨ 核心功能

### 🔐 用户认证系统
- **安全登录** - 支持用户名/密码登录，基于TCP协议的安全通信
- **账户管理** - 用户可以实时修改昵称和密码

### 👥 好友管理系统
- **智能搜索** - 支持通过用户ID或用户名搜索好友
- **好友申请** - 完整的好友请求-确认流程
- **好友列表** - 实时显示所有好友状态和在线情况
- **好友分组** - 清晰的联系人界面管理

### 💬 即时通讯功能
- **实时聊天** - 基于TCP的低延迟一对一聊天
- **聊天窗口** - 独立的聊天界面，支持多窗口聊天
- **消息记录** - SQLite本地数据库存储聊天历史
- **离线消息** - 自动接收和处理离线期间的消息
- **消息状态** - 实时显示消息发送和接收状态

### 📁 文件传输系统
- **文件发送** - 支持发送各种类型的文件
- **文件接收** - 自动保存到ReceivedFiles目录
- **传输管理** - 完整的文件传输状态跟踪
- **文件预览** - 支持文件信息显示和管理

### 🎨 用户界面
- **现代化设计** - 美观的粉色主题界面
- **响应式布局** - 左侧菜单栏+右侧内容区域
- **多页面管理** - 消息、联系人、个人信息等多个功能页面
- **自定义字体** - 集成自定义字体文件，提升视觉体验

## 🏗️ 技术架构

### 前端技术栈
- **Qt 5.15.2** - 跨平台GUI框架，提供原生界面体验
- **C++17** - 现代C++标准，确保代码性能和可维护性
- **TCP Socket** - 可靠的网络通信协议

### 后端支持
- **MySQL** - 稳定可靠的关系型数据库

## 🚀 快速开始

### 环境要求

- **Qt 5.15.2** 或更高版本
- **MinGW 32-bit** 编译器
- **MySQL 5.7+** 数据库
- **Windows 10/11** 操作系统

### 安装步骤

1. **克隆项目**
   ```bash
   git clone [https://github.com/little-heep/Chat-Client.git]
   cd ConnectionClient
   ```

2. **配置数据库**
   ```sql
   -- 创建数据库和必要的表结构
   CREATE DATABASE connection_db;
   -- 导入数据库结构文件
   ```

3. **编译项目**
   ```bash
   # 使用Qt Creator打开ConnectionClient.pro
   # 或使用命令行编译
   qmake ConnectionClient.pro
   make
   ```

4. **运行应用**
   ```bash
   ./ConnectionClient.exe
   ```

## 🎮 使用指南

### 首次使用

1. **启动应用** - 双击运行程序
2. **用户登录** - 输入用户名和密码进行登录
3. **主界面** - 登录成功后进入主聊天界面

### 主要操作

#### 👤 管理账户
- 点击设置按钮修改用户名或密码
- 系统会自动验证新信息的有效性

#### 🔍 添加好友
1. 使用搜索功能查找用户
2. 点击搜索结果中的用户
3. 发送好友申请
4. 等待对方确认

#### 💬 开始聊天
1. 在好友列表中选择联系人
2. 双击好友头像打开聊天窗口
3. 输入消息并发送
4. 支持表情、文件等多媒体内容

#### 📁 文件传输
1. 在聊天窗口中点击文件按钮
2. 选择要发送的文件
3. 确认发送，等待传输完成

## 🛠️ 开发指南

### 项目架构说明
- **mywindow.cpp/h** - 主窗口管理器，负责界面布局和页面切换
- **client.cpp/h** - 网络通信核心，处理TCP连接和消息收发
- **loginwidget.cpp/h** - 登录界面，处理用户身份验证
- **chatwidget.cpp/h** - 聊天窗口，实现实时消息交互
- **dbmanage.cpp/h** - 数据库管理器，处理本地数据存储
- **messagewidget.cpp/h** - 消息页面，管理聊天列表和好友搜索
- **allstructs.h** - 数据结构定义，包含所有通信协议结构

### 关键设计模式
- **信号槽机制** - Qt特有的事件处理机制，实现松耦合通信
- **MVC架构** - 分离界面、逻辑和数据层
- **组件化设计** - 每个功能模块独立封装，便于维护和扩展

### 扩展功能建议
项目采用模块化设计，便于添加新功能：
- **群聊系统** - 基于现有聊天框架扩展多人聊天
- **语音通话** - 集成音频编解码库
- **视频通话** - 添加摄像头和视频流处理
- **表情包系统** - 扩展消息类型支持图片表情
- **消息加密** - 添加端到端加密保护隐私
- **主题切换** - 扩展现有样式系统支持多主题

## 🐛 故障排除

### 常见问题

**Q: 无法连接到服务器**
- 检查网络连接
- 确认服务器是否正常运行
- 验证端口是否被防火墙阻止

**Q: 登录失败**
- 确认用户名和密码正确
- 检查数据库连接状态
- 查看服务器日志

**Q: 消息发送失败**
- 检查网络稳定性
- 确认好友在线状态
- 重启应用程序
